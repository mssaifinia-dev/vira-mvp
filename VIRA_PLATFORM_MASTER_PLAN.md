# دستور شروع هر جلسه جدید

اگر این سند در ابتدای گفتگو ارائه شد:

* پروژه را ادامه بده.
* هیچ معماری قبلی را تغییر نده مگر با دلیل فنی.
* قبل از هر کدنویسی ابتدا تحلیل، طراحی دیتابیس، طراحی API و طراحی UI را انجام بده.
* تمام توسعه ها باید در چارچوب Vira Platform انجام شوند.
* هدف نهایی ساخت یک Platform سازمانی قابل توسعه است، نه صرفاً یک فروشگاه اینترنتی.
